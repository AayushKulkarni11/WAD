Bootstrap Explanation for ass1a
================================

1. Project Overview
-------------------
This project is an ecommerce admin dashboard page built using:
- HTML
- Bootstrap 5
- Bootstrap Icons
- Custom CSS in `style.css`

Bootstrap is used here to create:
- page layout
- responsive grid
- navigation styling
- form styling
- cards
- table styling
- spacing
- text formatting
- shadow effects


2. Bootstrap Files Included
---------------------------
`<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">`
- This imports Bootstrap 5 CSS from a CDN.
- It gives access to Bootstrap classes such as `container-fluid`, `row`, `col-md-3`, `navbar`, `card`, `table`, and more.

`<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.10.5/font/bootstrap-icons.css">`
- This imports Bootstrap Icons.
- It allows icon classes such as `bi bi-house`, `bi bi-cart`, `bi bi-box`, `bi bi-people`, and `bi bi-gear`.


3. Bootstrap Classes Used in This File
--------------------------------------

1. `container-fluid`
- Creates a full-width container.
- It spans the full width of the screen.
- Used as the outer wrapper of the page.

2. `row`
- Creates a horizontal row in Bootstrap grid layout.
- Columns are placed inside a row.

3. `col-md-2`
- Creates a column that takes 2 grid units on medium screens and larger.
- Used for the sidebar.
- Bootstrap grid is based on 12 columns.

4. `col-md-10`
- Creates a column that takes 10 grid units on medium screens and larger.
- Used for the main content area.

5. `text-center`
- Aligns text to the center.
- Used for the sidebar heading and footer.

6. `mt-3`
- Adds top margin using Bootstrap spacing utility.
- `mt` means margin-top.
- `3` is the spacing size.

7. `nav`
- Styles the unordered list as a navigation container.

8. `flex-column`
- Makes the navigation items display in a vertical column.
- Uses flexbox direction column.

9. `mt-4`
- Adds top margin with a larger spacing size than `mt-3`.

10. `nav-item`
- Styles each list item as a navigation item.

11. `nav-link`
- Styles anchor tags as Bootstrap navigation links.

12. `bi`
- Base Bootstrap Icons class.
- Required before adding a specific icon name.

13. `bi bi-house`
- Displays a house icon.
- Used for the Dashboard menu item.

14. `bi bi-cart`
- Displays a cart icon.
- Used for the Orders menu item.

15. `bi bi-box`
- Displays a box icon.
- Used for the Products menu item.

16. `bi bi-people`
- Displays a people icon.
- Used for the Customers menu item.

17. `bi bi-gear`
- Displays a gear icon.
- Used for the Settings menu item.

18. `navbar`
- Applies Bootstrap navbar styling.
- Used for the top navigation bar in the main content area.

19. `bg-white`
- Sets the background color to white.
- Used in the navbar and table.

20. `p-3`
- Adds padding on all sides.
- `p` means padding.
- `3` is the spacing size.

21. `shadow-sm`
- Adds a small box shadow.
- Gives a soft elevated effect.

22. `mb-4`
- Adds bottom margin.
- `mb` means margin-bottom.

23. `form-control`
- Styles the input field using Bootstrap form design.
- Gives the search box proper width, padding, border, and look.

24. `w-25`
- Sets width to 25%.
- Used for the search input.

25. `fw-bold`
- Makes the text bold.
- `fw` means font-weight.

26. `col-md-3`
- Creates a column that takes 3 grid units on medium screens and larger.
- Four such columns fit in one row because `3 + 3 + 3 + 3 = 12`.
- Used for the dashboard cards.

27. `mb-3`
- Adds bottom margin below each column.
- Helps create spacing between card sections.

28. `card`
- Creates a Bootstrap card component.
- Used to show dashboard statistics in a boxed layout.

29. `card-body`
- Adds inner spacing and structure inside the card.

30. `table`
- Applies Bootstrap table styling.
- Makes the table cleaner and better formatted.

31. `table-bordered`
- Adds borders around the table and its cells.

5. Line-by-Line Bootstrap Usage in Layout
-----------------------------------------
`<div class="container-fluid">`
- Full-width page container.

`<div class="row">`
- Starts a grid row.

`<div class="col-md-2 sidebar">`
- Sidebar takes 2 columns on medium and larger screens.
- `sidebar` is custom CSS.

`<h3 class="text-center mt-3">`
- Heading is centered and given top margin.

`<ul class="nav flex-column mt-4">`
- Creates vertical navigation with top margin.

`<li class="nav-item">`
- Each menu option becomes a Bootstrap nav item.

`<a class="nav-link" href="#">`
- Styled navigation link.

`<i class="bi bi-house"></i>`
- Inserts Bootstrap icon for dashboard.

`<div class="col-md-10 main-content">`
- Main content takes 10 columns.
- `main-content` is custom CSS.

`<nav class="navbar bg-white p-3 shadow-sm mb-4">`
- Creates a white navbar with padding, small shadow, and bottom margin.

`<input type="text" placeholder="Search" class="form-control w-25">`
- Styled Bootstrap input box with 25% width.

`<span class="fw-bold">Welcome Admin</span>`
- Bold text for welcome message.

`<div class="col-md-3 mb-3">`
- Each card section takes 3 columns and has bottom margin.

`<div class="card card-box card1">`
- `card` is Bootstrap.
- `card-box` and `card1` are custom CSS classes.

`<div class="card-body">`
- Bootstrap card content area.

`<table class="table table-bordered bg-white">`
- Bootstrap table with borders and white background.

`<footer class="text-center mt-4">`
- Centered footer with top margin.


6. Why Bootstrap Is Used Here
-----------------------------
Bootstrap is used in this project because it:
- makes layout creation easier
- provides responsive grid system
- gives ready-made UI classes
- reduces the amount of custom CSS needed
- helps build clean pages quickly

In this dashboard, Bootstrap is mainly used for:
- responsive columns
- navigation
- spacing
- cards
- tables
- forms
- text alignment

8. Quick Summary of Main Bootstrap Classes
------------------------------------------
- `container-fluid` = full-width container
- `row` = horizontal grid row
- `col-md-*` = responsive grid columns
- `text-center` = center alignment
- `mt-*`, `mb-*`, `p-*` = spacing utilities
- `nav`, `nav-item`, `nav-link` = navigation styling
- `navbar` = navbar styling
- `bg-white` = white background
- `shadow-sm` = small shadow
- `form-control` = styled input
- `w-25` = 25% width
- `fw-bold` = bold text
- `card`, `card-body` = card component
- `table`, `table-bordered` = table styling
